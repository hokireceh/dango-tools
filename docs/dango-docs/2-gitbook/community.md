### hashtagHow to Get Involved


#### hashtagTrade on Alpha Mainnet
Visit dango.exchangearrow-up-right and create a Smart Account using your device&#x27;s passkey (no wallet extension required). Start with small amounts — this is an experimental Alpha release.


#### hashtagEarn Points
A points program is planned to launch alongside perpetuals trading. Points are expected to convert to tokens at TGE.


#### hashtagBuild on Dango
Dango is an open platform. Smart contracts are Rust/WASM modules running on Grug. The TypeScript SDK is available in the monorepo. If you&#x27;re a developer interested in building on Dango:


- Review the left-curve/left-curvearrow-up-right repo
- Read the Grug documentationarrow-up-right
- Engage with the team on X (@larry0xarrow-up-right)
