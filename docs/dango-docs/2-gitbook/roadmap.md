## hashtagRoadmap
Dango has moved through multiple testnet phases and launched Alpha Mainnet in January 2026. The following milestones reflect publicly announced and observed progress.


### hashtagCompleted Milestones


#### hashtagTestnet 1 — Core Trading

- On-chain CLOB with spot trading (ETH/USDC, BTC/USDC, SOL/USDC)
- Smart accounts with passkey authentication
- Subaccounts
- Basic spot order types (market, limit)

#### hashtagTestnet 1.5 — UX & Social Layer

- Native notification system
- Username system (human-readable account names)
- Block explorer integration
- UI improvements and performance optimizations

#### hashtagTestnet 2 — Advanced Features

- Vaults
- Lending
- Advanced order types
- Cross-chain bridging (Hyperlane integration)

#### hashtagAlpha Mainnet — January 2026

- Live on mainnet with real assets
- ETH/USDC spot pair active
- Unified margin account live
- Gasless USDC fee model active
- ⚠️ Experimental — use with small amounts only

### hashtagUpcoming Milestones


#### hashtagPerpetuals Launch (~March 2026)

- Perps trading with on-chain CLOB and funding rates
- Points program launch
- Additional spot pairs

#### hashtagMobile App

- Native iOS and Android apps
- Biometric (Face ID / fingerprint) authentication
- Full trading functionality on mobile

#### hashtagTGE — Q4 2026 (Planned)

- Token generation event
- Token listing and liquidity bootstrapping

#### hashtagPost-TGE

- Expanded validator set (path toward decentralization)
- Additional trading pairs and products
- Third-party app deployment on Dango chain
- Developer tooling and grants program[!NOTE] Timelines are approximate and based on publicly available information. Follow @dangoarrow-up-right on X for official announcements.
