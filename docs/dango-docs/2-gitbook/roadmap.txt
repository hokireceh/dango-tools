Roadmap

Dango has moved through multiple testnet phases and launched Alpha Mainnet in January 2026. The following milestones reflect publicly announced and observed progress.

hashtag

Completed Milestones

hashtag

Testnet 1 — Core Trading

On-chain CLOB with spot trading (ETH/USDC, BTC/USDC, SOL/USDC)

Smart accounts with passkey authentication

Subaccounts

Basic spot order types (market, limit)

hashtag

Testnet 1.5 — UX & Social Layer

Native notification system

Username system (human-readable account names)

Block explorer integration

UI improvements and performance optimizations

hashtag

Testnet 2 — Advanced Features

Vaults

Lending

Advanced order types

Cross-chain bridging (Hyperlane integration)

hashtag

Alpha Mainnet — January 2026

Live on mainnet with real assets

ETH/USDC spot pair active

Unified margin account live

Gasless USDC fee model active

⚠️ Experimental — use with small amounts only

hashtag

Upcoming Milestones

Additional spot pairs
