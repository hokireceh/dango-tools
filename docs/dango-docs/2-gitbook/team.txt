Team

Dango is built by  Left Curve Software , a small, technically deep team with roots in the Cosmos ecosystem.

hashtag

larry0x — Founder

X:

@larry0x  arrow-up-right

Veteran Cosmos contributor and smart contract developer

Previously worked on Mars Protocol and multiple CosmWasm-based applications

Author of the Grug execution environment

Known in the Cosmos community for high-quality, minimalist code and opinionated system design

The project&#x27;s technical direction reflects his philosophy of building bespoke infrastructure rather than adapting general-purpose tools

hashtag

Left Curve Foundation (LCF)

The  Left Curve Foundation  is the non-profit entity responsible for:

Selecting and managing the PoA validator set

Governing protocol upgrades

Overseeing DNG token distribution and treasury

Supporting ecosystem development and grants
